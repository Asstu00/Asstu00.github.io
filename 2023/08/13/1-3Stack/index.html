首先,介绍一下栈


这就是栈的样子(大概),可以想象为拆开的薯片桶,只有底但是没有顶
你往里放薯片,薯片只会落到桶底,然后你将桶装满,也只能优先取出装的最后一片薯片
栈取数据的过程,就类似于取薯片,你只能取最上面的一片(数据),而不能随意选取

栈有两种


满栈的初始指向,联系下文,想想为什么

1. 满递增栈

意思是 压栈(存数据) 时,flag(一种标志)指向 满栈(此处已存数据) ,每次需将flag指向上一格(高地址),然后再存数据
满递减栈类似
意思是 弹栈(取出数据) 时,flag指向满栈,每次需先弹栈(取数据),然后将flag指向下一格(低地址)


空栈的初始指向,联系下文,想想为什么

2. 空递增栈

意思是压栈时,flag指向空栈(此处无数据),每次需先存数据,然后将flag指向上一格
空递减栈类似
意思是弹栈时,flag指向空栈,每次需将flag指向下一格,然后再弹栈

总结,满栈的flag永远指向有数据的,空栈的flag永远指向无数据的

        return NULL;
    }
    array->top = -1;                                                // 该写法为满递增栈

}

void releaseArrayStack(ArrayStack *array) {
    if (array) {
        free(array);                        // 因为array的储存方式只有数组,所以直接释放就行

}

int popArrayStack(ArrayStack *array,element *elem) {
    if (array) {
        if (array.top < 0) {
            printf("UnderFlow\n");
            return -1;
