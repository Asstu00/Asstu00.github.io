1.4 Array Queue/Link Queue

  • C
  • linear
  • 数据结构

跟栈一样,两种写法,数组和链表

线性存储—队列

有两种存储方式,数组和链表

Array Queue

用数组存储,队头指向即将弹出的位置,队尾指向队尾元素的位置

由上面可以得到一个结构体
const int MaxSize = 5; // 定义数组最大范围
typedef struct {
element elem[MaxSize]; // element是需存储数据的类型,之前的文章出现过多次,以后不做解释
int front; // 队头
int rear; // 队尾
}ArrayQueue;

由上文所述,当rear指向 MaxSize-1 时,队列已满,但如果队列有元素弹出,front前的位置还有空余,此为假溢出
为了解决假溢出,我们将队列设计成循环队列

front = (front + 1) % MaxSize
rear = (rear + 1) % MaxSize
这样就成了循环队列

但这样又出现了新的问题,该如何判断队满和队空呢?
所以,我们约定
队空时
fornt == rear
队满时
(rear + 1) % MaxSize == front
这样会空余一个空间

逻辑行为

初始化,插入,删除,销毁

实现

  1. 初始化

    ArrayQueue *createArrayQueue() {
    ArrayQueue array = (ArrayQueue)malloc(sizeof(ArrayQueue));
    if (…) {…}
    array->front = 0;
    array->rear = 0;
    return array;
    }

  2. 销毁

    void releaseArrayQueue(ArrayQueue *array) {
    if(array){
    free(array);
    }
    }

  3. 插入

    pushArrayQueue(ArrayQueue *array, element elem) {
    if ((array->rear + 1) % MaxSize == array->front) { // 判断队列满没满
    printf(“ArrayQueue full!\n”);
    return -1;
    }
    array->rear = (array->rear + 1) % MaxSize;
    array->elem[array->rear] = elem;
    return 0;
    }

  4. 删除
    这里的删除直接覆盖就行

    int popArrayQueue(ArrayQueue *array, element *elem) {
    if (array->rear == array->front) {
    printf(“ArrayQueue empty”);
    return -1;
    }
    array->front = (array->front + 1) % MaxSize;
    *elem = array->elem[array-front];
    array->elem[array->front] = 0; // 将取走的元素变为0(方便观察qwq)
    return 0;
    }

Link Queue

链式存储就不用考虑那么多了

结构体换一下

// 节点
typedef struct node {
    element data;
    struct node* next;
}NodeQueue;
// 表头
typedef struct {
    NodeQueue *front;
    NodeQueue *rear;
    int len;
}LinkQueue;

实现

这里就直接写实现了,逻辑行为同上

    link->front = NULL:

    void releaseLinkQueue(LinkQueue *link) {
    while (link->front) {
    NodeQueue *tmp = link->front;
    link->front = tmp->next;
    free(tmp):
    link->len–;
    }
    printf(“queue have %d node!\n”, link->len);
    free(link); // 释放表头
    }

  3. 插入

    int pushLinkQueue(LinkQueue *link, element elem) {
    NodeQueue new_node = (NodeQueue)malloc(sizeof(NodeQueue));
    if (…) {…}
    new_node->data = elem;
    new_node->next = NULL;

    if (link->rear) { // 判断队列是否有元素
    link->rear->next = new_node;
    link->rear = new_node;
    } else {
    link->front = new_node;
    link->rear = new_node;
    }
    link->len++;
    return 0;
    }

    NodeQueue *tmp = link->front;
    link->front = tmp->next;
    free(tmp);
    link->len–;
    if (link->front == NULL) {
    link->rear = NULL;
    }
    return 0;
